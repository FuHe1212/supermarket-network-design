# 大型超市区域网设计方案
本仓库为大型超市区域网设计的毕业设计展示网站，基于HTML/CSS实现，包含网络拓扑、IP规划、设备清单、安全设计、测试方案等核心内容。

## 在线演示
https://fuhe1212.github.io/supermarket-network-design/

## 设计架构
- 核心层：核心路由器+核心交换机，负责全网数据转发和互联
- 汇聚层：汇聚交换机，负责接入层数据汇聚和业务隔离
- 接入层：接入交换机/无线AP，负责终端接入和PoE供电

## 核心功能
1. 多VLAN隔离不同业务区域，保障数据安全
2. 双机热备提升核心设备可用性
3. 分层安全策略，兼顾内网/外网/无线安全
4. 完善的测试方案，验证网络性能和稳定性
